## 📌 Project Overview
This project demonstrates how **Computer Vision (CV)** can be used in academic settings for **interactive learning**.  
It uses a camera to detect gestures, objects, or actions in real-time, making concepts easier to visualize and understand.  

**Purpose:**  
- Enhance classroom interactivity  
- Make exhibitions engaging  
- Demonstrate AI/CV concepts practically  

---

## 🎯 Objectives
- Engage students in interactive learning  
- Reduce dependency on traditional teaching methods  
- Demonstrate real-time gesture/object detection  
- Make academic exhibitions visually appealing and functional  

---

## 🧠 Technologies Used
- **Python 3.11+**  
- **OpenCV** – Computer Vision library  
- **MediaPipe** – Hand/pose detection  
- **NumPy** – Matrix computations  

---

## ⚙️ Features
- Real-time camera detection  
- Hand/gesture-based interaction  
- Interactive academic demonstrations  
- Easy to run and understand  
- Suitable for exhibitions, workshops, and classroom demonstrations
